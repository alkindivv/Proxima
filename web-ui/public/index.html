🎯 Core AI Tools

/council "message" --mode [consensus|debate|critique] --files [...] --context [...]
Multi-AI deliberation across all green providers. Modes: consensus (default) = gather & synthesize; debate = structured debate with position/challenge/moderator; critique = critical review finding flaws & risks.
/council "Should I trade EURUSD or XAUUSD?" --mode debate
/ask_all_ais "question"
Ask all enabled AI providers the same question simultaneously.
/ask_all_ais "What is the best timeframe for scalping?"
/compare "question" --providers [p1,p2...]
Compare responses from specific providers. Shows individual results + aggregate.
/compare "EURUSD outlook" --providers perplexity,chatgpt,gemini

🔍 Research Tools

/deep_search "query" --files [paths]
Deep research with file context support. Includes code/documents.
/deep_search "gold price analysis 2024" --files /path/to/data.csv
/internet_search "query"
Search the internet for current, up-to-date information.
/internet_search "Federal Reserve interest rate decision"
/news_search "topic" --timeframe [today|week|month]
Search latest news on a topic.
/news_search "EURUSD" --timeframe this week
/github_search "query" --language [lang]
Search GitHub for repos, code examples, and libraries.
/github_search "trading bot Python" --language Python
/reddit_search "query"
Search Reddit discussions.
/reddit_search "scalping strategies forex"

💻 Code Tools

/generate_code "description"
Generate code in any language based on description.

/review_code "code" --context [info]
Review code for bugs, security issues, and best practices.
/review_code "def calculate_pip(): ..." --context "forex trading function"
/optimize_code "code"
Optimize code for performance and efficiency.
/optimize_code "for i in range(1000): print(i)"
/explain_code "code"
Explain what code does in plain language.
